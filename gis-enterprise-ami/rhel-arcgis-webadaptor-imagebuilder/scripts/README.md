# Scripts Directory

This directory contains various scripts used for AMI building and management, organized by scripting language.

## Structure

- `shell/` - Bash/Shell scripts for Linux/Unix operations
- `python/` - Python scripts for automation and utilities

## Usage

Each subdirectory should contain scripts specific to that language/platform. Include proper documentation and error handling in all scripts.

## Best Practices

- Use descriptive filenames
- Include shebang lines in scripts
- Add proper error handling and logging
- Document script parameters and usage
- Make scripts executable with appropriate permissions
