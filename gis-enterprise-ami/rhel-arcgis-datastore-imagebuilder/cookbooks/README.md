# Cookbooks Directory

This directory contains configuration management cookbooks and playbooks for infrastructure automation.

## Structure

- `cinc/` - Chef cookbooks and recipes

## Usage

### cinc
Place Chef cookbooks in the `cinc/` directory. Each cookbook should follow standard Chef directory structure.


## Best Practices

- Use version control for all cookbooks
- Test configurations in development environments first
- Document all variables and parameters
- Follow naming conventions for consistency
- Use encrypted storage for sensitive data
